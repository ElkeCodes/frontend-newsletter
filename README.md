# Newsletter Renderer

An Astro-powered project that converts markdown newsletters into clean, email-ready HTML pages.

## Features

- 📝 **Markdown to Email**: Convert markdown files to beautifully formatted emails
- 🎨 **Clean Styling**: Minimal CSS with rem-based spacing and sizing
- 🌙 **Dark Mode**: Automatic light/dark theme support using CSS variables
- 📋 **Frontmatter Support**: Define metadata (title, subject, date, author) in YAML frontmatter
- 🧱 **Semantic HTML**: Uses `article`, `header`, `main`, `footer`, `time`, and `address`
- 📱 **Responsive**: Mobile-friendly email design

## Installation

```bash
npm install
```

## Usage

### Basic Usage

```bash
npm run render
```

This builds static HTML files into `dist/`.

### Specify Output File

Rendered newsletter pages are generated by Astro routes. The sample newsletter is available at:

- `dist/newsletters/sample/index.html`

### Development

Start Astro in development mode:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Preview the built output:

```bash
npm run preview
```

## Markdown Format

Create a markdown file in `src/pages/newsletters/` with YAML frontmatter at the top:

```markdown
---
layout: ../../layouts/NewsletterLayout.astro
title: My Newsletter Title
subject: Email Subject Line
date: 2024-01-15
author: Your Name
preview: Optional preview text for email clients
---

# Main Content

Your markdown content goes here...
```

### Metadata Fields

- **layout** (required): `../../layouts/NewsletterLayout.astro`
- **title** (required): Newsletter title
- **subject** (required): Email subject line
- **date** (optional): Publication date (defaults to today)
- **author** (optional): Author name
- **preview** (optional): Preview text for email clients

## Supported Markdown

Astro markdown rendering supports standard markdown formatting:

- **Headings**: `# H1`, `## H2`, `### H3`
- **Text styles**: `**bold**`, `*italic*`
- **Lists**: Unordered and ordered lists
- **Links**: `[text](url)`
- **Code blocks**: Inline and fenced code blocks
- **Blockquotes**: `> quote text`

## Output

The build generates complete HTML newsletters with:

- Professional header with title and metadata
- Responsive container (max-width: 600px)
- Clean typography optimized for email clients
- Semantic document structure
- Automatic dark mode styling
- Basic footer with disclaimer
